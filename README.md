# Diamond Pattern With Alphabets in C#

### This program will print you characters in below format.

Examples

    > DiamondPatternWithAlphabets.exe A
      A

    > DiamondPatternWithAlphabets.exe B
       A
      B B
       A

    > DiamondPatternWithAlphabets.exe C
        A
       B B
      C   C
       B B
        A
